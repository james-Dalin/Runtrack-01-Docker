#Job01 

##Commande exécuté 
```
docker run hello-world
```

##Ce qui se passe

1. Le client Docker contacte le daemon Docker
2. Le daemon télécharge l'image 'hello-world' depui le Docker Hub
3. Le daemon crée un conteneur à partir de cette image
4. Le conteneur affiche le message de confirmation et s'arrête
